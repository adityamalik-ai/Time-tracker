Connect your Supabase database
One-time setup. Your Supabase project is the shared backend — employees write tracking data there, you read it in real-time from the dashboard.
  1. Go to supabase.com → New project → copy Project URL & anon public key
  2. In Supabase → SQL Editor → run the schema below once
  3. Paste your credentials here and click Save
-- Run this once in Supabase SQL Editor
create table if not exists pw_employees (
  email text primary key,
  name text,
  data jsonb,
  updated_at timestamptz default now()
);
create table if not exists pw_credentials (
  email text primary key,
  pass text,
  name text
);
alter table pw_employees enable row level security;
alter table pw_credentials enable row level security;
create policy "anon all employees" on pw_employees for all using (true) with check (true);
create policy "anon all credentials" on pw_credentials for all using (true) with check (true);
Credentials stored in localStorage. Employees never see this screen.
